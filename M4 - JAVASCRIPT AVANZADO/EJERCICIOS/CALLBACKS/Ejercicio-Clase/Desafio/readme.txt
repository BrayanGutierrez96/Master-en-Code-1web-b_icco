Desafio:
Convertidor de Temperatura con Callback

Objetivo:
Crear una aplicación web que convierta temperaturas entre grados Celsius y Fahrenheit utilizando una función de callback para procesar y mostrar los resultados.

Requerimientos:
Crear un archivo HTML con un campo de entrada, un botón para realizar la conversión, y un lugar para mostrar el resultado.

El usuario debe poder convertir de Celsius a Fahrenheit y viceversa.

Utilizar una función de callback para realizar la conversión y actualizar la interfaz de usuario.
