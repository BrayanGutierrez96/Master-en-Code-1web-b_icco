Desafio: mis libros favoritos

Crea una clase llamada Libro que represente un libro. La clase debe tener los siguientes atributos: titulo, autor, añoPublicacion y genero. Además, debe tener un método llamado mostrarDetalles() que imprima por consola todos los detalles del libro, ademàs tendràs que agregarlos a un arreglo, para mostrar todos los libros que se creen mediante instancias.

Instrucciones:

1. Define una clase llamada Libro.
2. La clase debe tener cuatro atributos: titulo, autor, anioPublicacion y genero, todos ellos son cadenas de texto excepto anioPublicacion, que es un número.
3. Define un constructor que tome cuatro parámetros y los utilice para inicializar los atributos de la clase.
4. Define un método llamado mostrarDetalles() que imprima por consola todos los detalles del libro en un formato legible.
5. Crea al menos dos instancias de la clase Libro con diferentes detalles.
6. Crea un arreglo llamado mostrarListaLibros que me imprima en consola todos los libros que hemos creado con la clase.
7. Utiliza el método mostrarDetalles() en cada instancia para mostrar los detalles de cada libro.
