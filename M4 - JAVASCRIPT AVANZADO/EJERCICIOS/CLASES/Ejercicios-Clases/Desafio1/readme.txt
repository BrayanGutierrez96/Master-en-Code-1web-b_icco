Desafío: Calcula mi area

Crea una clase llamada Rectángulo que represente un rectángulo. La clase debe tener los siguientes atributos: ancho y altura. Debe tener un método llamado calcularArea() que calcule y devuelva el área del rectángulo.

Instrucciones:

1. Define una clase llamada Rectángulo.
2. La clase debe tener dos atributos: ancho y altura, ambos números.
3. Define un constructor que tome dos parámetros, ancho y altura, y los utilice para inicializar los atributos de la clase.
4. Define un método llamado calcularArea() que calcule el área del rectángulo (ancho * altura) y lo devuelva.
5. Crea dos instancias de la clase Rectangulo con diferentes valores de ancho y altura.
6. Utiliza el método calcularArea() en cada instancia para calcular y mostrar el área de cada rectángulo.
